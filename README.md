## Pasos para la iniciación de los ejercicios 
### Requisitos previos 
- Tener instalado **Node.js**, que incluye el gestor de paquetes **npm**.
- Puedes comprobarlo ejecutando:

  ```bash
   node -v npm -v
  ```

  ## Instalación

  Clona o descarga el repositorio. Desde la carpeta raíz del proyecto, ejecuta el siguiente comando para instalar las dependencias:
  ```bash
  npm install
  ```
